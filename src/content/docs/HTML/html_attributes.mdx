1. Global attributes apply to ALL HTML elements
2. Other attributes only apply to a number of HTML elements

## Global attributes

- id
- class
- style

## Other common attributes

- src
- href
- width
- height
- alt

## How to modify attributes with javascript

```html title="index.html"
<img src="anzac.png"/>
```

```js title="index.js"
let element = document.querySelector("img");
element.src = "biscotti.png";
```
