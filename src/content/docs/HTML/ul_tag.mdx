1. ul (unordered list) contains multitple li(list items)
2. Used heavily for menus and sidebars
3. By default, they add bullet points, remove with  ```list_style="none"``` 

```js
<!-- Basic unordered list, contains bullet points -->
<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Cherries</li>
</ul>

<!-- Remove bullets -->
<ul style="list-style-type: none;">
  <li>No bullet 1</li>
  <li>No bullet 2</li>
</ul>

<!-- With custom icons as bullets -->
<ul style="list-style: none;">
  <li>🍎 Apple</li>
  <li>🍊 Orange</li>
  <li>🍌 Banana</li>
</ul>
```
