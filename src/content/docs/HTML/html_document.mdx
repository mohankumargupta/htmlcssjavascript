1. A webpage is described with a **HTML document**.

2. The document is of type **HTML5** document .
   <HTML5Icon display="inline !important" font-size="3rem"/>

## Minimal HTML 5 document

```html title="index.html"
<!DOCTYPE html>
<html>
<body>
    <h1>Hello World</h1>
    <p> This html document is missing a head element, thats OK.<p>
</body>
</html>
```

<Aside>
The &lt;!DOCTYPE html&gt; declaration tells the browser that what follows is a HTML 5 document
</Aside>

## HTML with internal CSS and JS
```html title="index.html"
<!DOCTYPE html>
<html>
<head>
    <title>Hello World Title</title>
    <style>
        h1 {
            color: red;
        }
    </style>
    <script>
       function button_clicked() {
          const p = document.querySelector("#para");
          p.textContent = "Button clicked!";
       }
    </script>
</head>
<body>
    <h1>Hello World</h1>
    <p id="para"></p>
    <button onclick="button_clicked()">Click me</button>
</body>
</html>
```

## Access document in javascript

Better to use this when walking the DOM tree 

```js title="console-dir-document.js"
console.dir(document);
```

Than this one

```js title="console-log-document.js"
console.log(document);
```

```js title="title.js"
const title = document.title;
console.log(title);
```
