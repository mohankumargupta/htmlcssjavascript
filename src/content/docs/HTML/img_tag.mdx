```js
<!-- Basic image -->
<img src="photo.jpg" alt="Description of image">

<!-- Set width and height -->
<img src="photo.jpg" alt="Resized" width="200" height="150">

<!-- Link an image -->
<a href="https://example.com">
  <img src="logo.png" alt="Example Logo">
</a>
```
