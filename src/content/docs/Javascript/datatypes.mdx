## Data Types
1. A value has a data type
2. There are 8 data types, 7 of them are primitive data types.

### 7 Primitive Data Types
Primitive means simple. In the javascript context, means represents a single value.

1. number
2. string
3. boolean
4. null
5. undefined

There are two other less used data types. 

![typeof](https://user-images.githubusercontent.com/29270715/72416480-d9672f00-379e-11ea-83d5-2e7219d1594f.png)

<Aside>
Let us try to find out what the data type of values are.

Open your browser's console(Right click->Inspect->Console).


```js
console.log(typeof(2));           // number
console.log(typeof("hello"));     // string
console.log(typeof(true));        // boolean
```
</Aside>


### Object data type

1. An object let's us create are own data type.
2. When a primitive data type won't do the job, create an object.
3. An object lets us create a more complex data type from the primitive data types.
