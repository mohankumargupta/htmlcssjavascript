1. Values in a computer program represent some useful data the program is working with.
2. They are fixed, can't be changed.
3. Values belong to a data type.

<Aside>
The important primitive data types: numbers, strings, booleans. 

Open your browser's console(Right click->Inspect->Console).

Print these primitive values using console.log():

```js
console.log(2);           // This is a number
console.log("hello");     // This is a string
console.log(true);        // This is a boolean
```
</Aside>
