<Aside type="tip">
To react on events we can assign an event handler, a function that runs when event occurs.

Two ways to do this.

First way:  uses on[event] attribute on HTML element.

```html
<button onclick="myFunction()">Click me</button>
```

Second way: In JavaScript click is the event, myFunction is the event handler.

```js
button.addEventListener("click", myFunction);
```
</Aside>
