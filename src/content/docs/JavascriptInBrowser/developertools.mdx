Important to be familiar with Inspect(Developer) tools in your favourite browser.

The core functionality is roughly the same between Edge, Chrome and Firefox.

## Open Developer Tools

Right click->Inspect

It should open at the bottom, if it opens to the side:

1. Click the 3 dots
2. Change **dock side**

## What can you do with the Developer Tools
1. Inspect and change live HTML (Elements tab)
2. Inspect and change live CSS (Elements tab)
3. Inspect javascript (Sources tab)
4. Run arbitrary javascript (Console tab)
5. Debug javascript by putting breakpoints
6. See how resouces are loaded (Network tab)
